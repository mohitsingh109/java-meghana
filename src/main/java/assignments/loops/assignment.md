
# Java Loops MCQ Assignment

### Question 1
What will the following code print?

```java
int x = 0;
for (int i = 1; i <= 5; i++) {
    x += i;
}
System.out.println(x);
```

1. `10`
2. `15`
3. `20`
4. `25`

### Question 2
What will the following code print?

```java
int i = 5;
while (i > 0) {
    System.out.print(i + " ");

}
```

2. `5 4 3`
3. `5 2`
4. `5 3`
